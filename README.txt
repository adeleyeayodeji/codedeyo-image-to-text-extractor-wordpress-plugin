=

* Can extract text from image.

== Installation ==

1. Upload `codedeyo-image-to-text-extractor` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Require 'WordPress classic editor' to work perfectly
